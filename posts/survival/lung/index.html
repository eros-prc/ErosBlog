2.5 插值

2.5.1 补充知识：

  1. 一般使用pmm是最好的选择
  1. 数据存在强非线性 / 复杂交互效应 当变量间关系高度非线性（如 U 型关系）或存在复杂交互作用时，随机森林（rf） 或CART（cart） 可能更优，它们能捕捉 PMM 无法建模的非线性模式。 适用案例：肿瘤大小与生存时间呈非线性关系，且与治疗方式存在交互效应
  2. 分类变量插值 PMM 仅适用于连续变量，分类变量必须选择对应方法： 二分类 → logreg（逻辑回归） 无序多分类 → polyreg（多项式回归） 有序多分类 → polr（比例优势模型）
  3. 高维稀疏数据（如基因表达数据） 当变量数远大于样本数（p >> n）时，PMM 的回归模型可能不稳定，此时可考虑： rf（随机森林，对高维数据有天然优势） lda（线性判别分析，降维后插值）
  4. 特殊研究设计（如加权抽样、纵向数据） 加权抽样数据 → midastouch（加权 PMM） 纵向数据 → mice的2l.norm（两水平正态模型）等专门方法

因此本文选择如下方法：

  • 连续变量（meal.cal, wt.loss）：pmm

  • 分类变量（如 sex, ph.ecog）：logreg/删除

3.3 cox假设检验（PH检验）

风险比假设：不同组之间的风险比 HR 随时间保持不变。

若假设不成立：

  • 如果变量不是主要研究变量，可以保留并说明局限

  • 加入时间交互项

  • 分层 Cox 模型

ph_results <- map_dfr(1:imp$m, function(i) {

## 插值

### 补充知识：

0.  一般使用pmm是最好的选择

<!-- -->

1.  数据存在强非线性 / 复杂交互效应 当变量间关系高度非线性（如 U 型关系）或存在复杂交互作用时，随机森林（rf） 或CART（cart） 可能更优，它们能捕捉 PMM 无法建模的非线性模式。 适用案例：肿瘤大小与生存时间呈非线性关系，且与治疗方式存在交互效应
2.  分类变量插值 PMM 仅适用于连续变量，分类变量必须选择对应方法： 二分类 → logreg（逻辑回归） 无序多分类 → polyreg（多项式回归） 有序多分类 → polr（比例优势模型）
3.  高维稀疏数据（如基因表达数据） 当变量数远大于样本数（p \>\> n）时，PMM 的回归模型可能不稳定，此时可考虑： rf（随机森林，对高维数据有天然优势） lda（线性判别分析，降维后插值）
4.  特殊研究设计（如加权抽样、纵向数据） 加权抽样数据 → midastouch（加权 PMM） 纵向数据 → mice的2l.norm（两水平正态模型）等专门方法

因此本文选择如下方法：

-   连续变量（meal.cal, wt.loss）：pmm

-   分类变量（如 sex, ph.ecog）：logreg/删除
