# Eros Blog

这是一个基于 Quarto 的个人博客项目，主要用于整理统计学、数据分析、数学建模、编程学习、哲学阅读和个人日志。

## 主要目录

```text
posts/       正式文章
assets/      CSS、JS、图片、数据等公共资源
drafts/      草稿区
docs/        项目说明和维护文档
_templates/ 文章模板
```

详细说明见：

```text
docs/博客目录说明.md
docs/日志维护说明.md
docs/发布流程.md
docs/趣味部件维护说明.md
```

## 常用命令

```bash
quarto preview
quarto render
quarto publish gh-pages
```
