# 趣味部件维护说明

本项目在首页和全站增加了几个轻量互动部件，主要文件如下：

```text
index.qmd                         首页结构
assets/css/styles.css             首页卡片、Bento、进度条等样式
assets/js/site-widgets.js         随机语录、随机文章、阅读进度条、首页心情预览
assets/js/site-widgets-loader.html 全站脚本加载器
```

## 已加入的部件

1. 首页 Hero 区域
2. 当前状态卡片
3. 今日一句随机语录
4. Mood Journal 首页小预览
5. 学习进度条
6. Bento Grid 内容导航
7. 标签云
8. 随机文章按钮
9. 全站阅读进度条

## 修改随机语录

打开：

```text
assets/js/site-widgets.js
```

找到：

```js
const quotes = [ ... ];
```

直接增删句子即可。

## 修改随机文章按钮

打开：

```text
assets/js/site-widgets.js
```

找到：

```js
const randomLinks = [ ... ];
```

新增文章时，可以按下面格式加一条：

```js
{ title: "文章标题", href: "posts/分类/文章目录/" }
```

## 修改学习进度条

打开首页：

```text
index.qmd
```

找到：

```html
<i style="--p: 78%"></i>
```

把百分比改掉即可。

## 修改首页分类卡片

打开：

```text
index.qmd
```

找到：

```html
<div class="bento-grid">
```

在里面修改卡片标题、说明和链接。

## 注意

- 这些部件都是静态网页前端功能，不需要后端。
- 如果发布到 GitHub Pages，建议先运行 `quarto preview` 确认页面正常。
- 如果随机文章按钮跳转不对，优先检查 `assets/js/site-widgets.js` 里的 `href` 是否写对。
