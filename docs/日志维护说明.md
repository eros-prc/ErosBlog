# 日志与心情热力图维护说明

## 1. 更新每日心情

打开：

```text
assets/data/mood.csv
```

每天新增一行：

```csv
2026-07-01,4,3,今天状态还不错,生活|学习
```

字段顺序是：

```text
date,mood,energy,title,tags
```

- `mood`：心情分数，1 到 5。
- `energy`：精力分数，1 到 5。
- `title`：当天一句话标题。
- `tags`：标签，用 `|` 分隔。

## 2. 写公开日志

复制模板：

```text
_templates/journal-day-template.qmd
```

然后放到：

```text
posts/journal/2026-07-01/index.qmd
```

如果只是自己记录，不想公开，就不要放到 `posts/`。

## 3. 写私密日志

建议放在本地：

```text
private-journal/
```

这个文件夹已经写进 `.gitignore`，不建议上传到公开 GitHub 仓库。

## 4. 修改解锁密码

打开：

```text
assets/js/mood-heatmap.js
```

找到：

```js
const JOURNAL_PASSWORD = "eros2026";
```

把 `eros2026` 改成自己的密码即可。

注意：这个只是静态页面前端隐藏，不是真正后端登录，不要放非常隐私的内容。
