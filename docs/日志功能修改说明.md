# 日志功能修改说明

本项目已经新增日志板块，并按照整理后的目录结构存放相关文件。

## 主要新增内容

1. `journal.qmd`：日志首页。
2. `assets/data/mood.csv`：心情热力图数据。
3. `assets/js/mood-heatmap.js`：读取 CSV 并生成热力图。
4. `assets/css/styles.css`：全站样式和日志页样式。
5. `_templates/journal-day-template.qmd`：每日日志模板。
6. `private-journal/`：本地私密日志说明，已加入 `.gitignore`。

## 维护重点

- 每天更新心情：改 `assets/data/mood.csv`。
- 修改密码：改 `assets/js/mood-heatmap.js` 中的 `JOURNAL_PASSWORD`。
- 真正隐私内容不要上传到公开 GitHub 仓库。
