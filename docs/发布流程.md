# Quarto Blog 发布流程

## 本地预览

在博客根目录运行：

```bash
quarto preview
```

## 渲染检查

```bash
quarto render
```

## 发布到 GitHub Pages

```bash
quarto publish gh-pages
```

## 常见提醒

1. 正式文章放到 `posts/` 对应栏目。
2. 草稿放到 `drafts/`。
3. 公共图片放 `assets/images/`。
4. 某篇文章自己的图片直接放在该文章文件夹。
5. 心情热力图数据更新 `assets/data/mood.csv`。
6. 不要把真正隐私内容上传到公开仓库。
