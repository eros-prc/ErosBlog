# 博客目录说明

这版整理后的目录按照「页面、文章、资源、模板、文档、草稿」分开，后续维护会更清楚。

```text
blog/
├─ _quarto.yml                 # Quarto 网站总配置
├─ index.qmd                   # 首页
├─ about.qmd                   # 关于页面
├─ statistics.qmd              # 统计学基础栏目页
├─ modeling.qmd                # 统计建模栏目页
├─ survival.qmd                # 生存分析栏目页
├─ r.qmd                       # R 语言栏目页
├─ python.qmd                  # Python 栏目页
├─ causal.qmd                  # 因果推断栏目页
├─ math-modeling.qmd           # 数学建模栏目页
├─ journal.qmd                 # 日志 / 心情热力图页面
├─ althusser.qmd               # 阿尔都塞栏目页
├─ philosophy.qmd              # 柏拉图栏目页
│
├─ posts/                      # 正式发布的文章
│  ├─ statistics/
│  ├─ modeling/
│  ├─ survival/
│  ├─ r/
│  ├─ python/
│  ├─ causal/
│  ├─ math-modeling/
│  ├─ althusser/
│  └─ philosophy/
│
├─ assets/                     # 全站公共资源
│  ├─ css/styles.css           # 全站样式
│  ├─ js/mood-heatmap.js       # 日志热力图脚本
│  ├─ data/mood.csv            # 心情数据
│  └─ images/                  # 全站图片
│     ├─ avatar/profile.jpg
│     └─ site/
│
├─ _templates/                 # 可复制的写作模板
├─ drafts/                     # 草稿区，不会自动进入文章列表
├─ private-journal/            # 私密日志说明，本地使用，不建议上传
├─ docs/                       # 项目维护说明
└─ _archive/                   # 暂存旧文件、备用文件
```

## 新文章放哪里

建议所有正式文章都使用下面这种结构：

```text
posts/栏目名/文章英文短名/index.qmd
posts/栏目名/文章英文短名/image.png
```

例如：

```text
posts/python/pandas-basic/index.qmd
posts/python/pandas-basic/image.png
```

这样每篇文章的文字、图片、附件都在同一个文件夹里，最不容易乱。
