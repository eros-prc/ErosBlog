# 日志热力图修复说明

## 为什么之前看不出波动

之前使用的是黑白灰渐变，`未记录` 与 `心情 1 分` 的颜色太接近。再加上一年有 365 个格子，而目前只记录了几天，所以视觉上会觉得“几乎没有变化”。

现在已改成高对比颜色：

- 1 分：红色
- 2 分：橙色
- 3 分：黄色
- 4 分：浅绿色
- 5 分：深绿色
- 未记录：极浅灰色

同时新增了“心情波动线”，记录 2 天以上后会自动显示折线。

## 连续打卡怎么算

连续打卡不是手动输入的，而是脚本根据 `assets/data/mood.csv` 自动计算的。

规则是：从最新一条记录日期开始，往前一天一天检查，只要每天都有记录就继续加 1；中间缺一天就停止。

例如：

```csv
2026-06-28,1,1,普通一天,生活
2026-06-29,2,2,稍好一点,学习
2026-06-30,3,3,状态恢复,博客
```

连续打卡 = 3 天。

如果是：

```csv
2026-06-28,1,1,普通一天,生活
2026-06-30,3,3,状态恢复,博客
```

因为缺少 `2026-06-29`，连续打卡 = 1 天。

## 每天怎么添加

打开：

```text
assets/data/mood.csv
```

按下面格式追加一行：

```csv
2026-07-01,3,3,今天状态还可以,博客|学习
```

完整表头必须保留：

```csv
date,mood,energy,title,tags
```
