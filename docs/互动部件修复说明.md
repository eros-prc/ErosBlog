# 互动部件修复说明

这版主要修复首页内容导航卡片显示 `<div>`、`<h3>`、`<p>` 源码的问题。

## 修复原因

之前首页把卡片内容直接写成原生 HTML。部分 Quarto / Pandoc 渲染环境会把这些标签转义成普通文本，所以页面上会显示源码。

## 现在的做法

首页 `index.qmd` 只保留 Quarto 原生 Markdown 和几个空容器，具体内容由：

```text
assets/js/site-widgets.js
```

在浏览器中动态生成。这样不会再出现 HTML 标签被当成文字显示的问题。

## 新增互动部件

首页新增了：

1. 心情生成器
2. 学习抽签
3. 25 分钟番茄钟
4. 知识翻牌卡
5. 一键专注模式

7. JS 生成的标签云

## 使用提醒

建议不要把新版压缩包直接“合并覆盖”到旧文件夹里。更稳的方式是：

1. 先备份旧博客文件夹。
2. 解压这个新版压缩包到一个空目录。
3. 在新目录运行 `quarto preview`。
4. 确认没问题后再发布。

如果浏览器仍然显示旧样式，按 `Ctrl + F5` 强制刷新，或者清理浏览器缓存。
